Where to place JS scripts and why?

JS scripts are placed inside script tag in head or body tag.
It is recommended to place the script at the end of the body element in case it is necessary to access any element that contains the body and to improve the page load time.

CSS

Explain CSS Grid and Flexbox including use-cases.

Grid is a CSS property that is given to an element to establish the number of rows and columns that is desired and the children defines the space they occupy within said grid. It is used when you need to define a fixed layout.

Flexbox is a CSS property that sets flex elements along the horizontal or vertical axis, requiring you to decide whether you want to set rows or columns. It is used when you need to define flexible child elements to the container.

Explain the difference between em & rem.

They are two units of relative CSS measurements.
The difference is that em references the font size of the parent element and rem references the font size of the root (html) element.
